# **App Name**: CodeVertex Portfolio

## Core Features:

- Dynamic Hero Section: Display full name, tagline, animated skills, and relevant buttons.
- Interactive About Me Section: Showcase intro, skills badges, and an education timeline.
- Categorized Skills Section: Display programming, web, tools, and cybersecurity skills using animated progress bars or cards.
- Dynamic Project Cards: Display projects with image thumbnail, title, tech stack, description, and links to GitHub and live demo.
- Certification Timeline: Present certifications and achievements in a timeline or card layout.
- Contact Form with Supabase Integration: Capture user messages and store them securely in Supabase with email validation.
- Secure Admin Panel: Hidden admin page accessible via /admin, secured with Firebase Authentication, allowing message viewing, deletion, and logout.

## Style Guidelines:

- Primary color: Neon blue (#00FFFF) to create a vibrant and modern tech aesthetic.
- Background color: Dark gray (#222222) for a sleek developer-themed base.
- Accent color: Electric purple (#BF00FF) for highlights and interactive elements.
- Body and headline font: 'Space Grotesk', sans-serif for headings and 'Inter', sans-serif for body to ensure readability and a modern aesthetic.
- Use minimalist, neon-style icons to represent different skills and technologies.
- Employ a grid-based layout with glassmorphism cards to achieve a clean, modern UI.
- Incorporate smooth scrolling animations and loading animations to enhance user experience.
